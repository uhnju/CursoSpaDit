Generando HTML en Diferentes Enfoques

Introducción

En el desarrollo web moderno, existen diversas maneras de generar y manipular elementos HTML mediante código...

1️⃣ JavaScript puro (sin React)

const Example = () => {
  const div = document.createElement("div");
  div.textContent = "It's Me. Hi!";
  div.className = "example";
  return div;
};
document.body.appendChild(Example());

2️⃣ React sin JSX

const Example = () => {
  return React.createElement("div", { className: "example" }, "It's Me. Hi!");
};

3️⃣ React con JSX

const Example = () => {
  return 
It's Me. Hi!

};

📌 Comparación y Conclusión

Enfoque Ventajas Desventajas
JavaScript puro No depende de bibliotecas externas Manipular el DOM puede ser tedioso
React sin JSX No requiere transformación Sintaxis más verbosa y difícil de leer
React con JSX Código más legible y limpio Requiere Babel para compilar JSX
